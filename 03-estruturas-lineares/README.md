# Estruturas Lineares

Pilhas, filas, deques e listas são exemplos de coleções de dados lineares, ou seja, os elementos presentes na estrutura estão ligados entre si sem que haja lacunas, desvios ou múltiplas ligações. Estas estruturas possuem regras rígidas para realizar as operações  (inserção, remoção, acesso, etc). A principal diferença entre estas estruturas é a maneira como estes itens são posicionados nas operações, em particular a extremidade onde ocorrem. 

Estas estruturas são usadas em diversas aplicações, em muitos algoritmos e usadas ​​para resolver uma variedade de problemas importantes. Ao final de cada módulo, você aplicará o que aprendeu desenvolvendo projetos práticos, como: conversão de números entre bases diferentes, editor de texto com histórico de edições, sistema de controle de fila de banco,  um carrinho de compras, e uma agenda de contatos.
